# Getting Started

Welcome to the 6eDesign Framework! This guide will help you set up your first project.

## Installation

To get started, ensure you have Node.js (v18+) and pnpm installed.

```bash
pnpm create svelte@latest my-app
cd my-app
pnpm install
```
